\doxysubsection{Detailed Description}
Implementacao das funcoes para manipular as antenas e os efeitos nefastos. 

\begin{DoxyAuthor}{Author}
Matheus Delgado (a31542 IPCA)
\end{DoxyAuthor}
Este ficheiro contem as funcoes para criar, inserir, remover e alterar antenas e cacular os efeitos nefastos entre antenas com a mesma frequencia. As funcoes tambem incluem a gravacao e leitura de dados em ficheiros binarios e de texto para representacao grafica
